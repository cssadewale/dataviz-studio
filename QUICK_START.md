# Tableau Stimulator — Quick Start

1. Open `index.html` in a browser, or deploy the repository to GitHub Pages.
2. Click **Sample Data** to load the built-in sales dataset, or upload your own CSV/TSV.
3. Drag a dimension to **COLS** and a measure to **ROWS**.
4. Click **Render**, or use **Show Me** for a Tableau-style suggestion.
5. Use **Ask Data** for simple local questions such as `top sales by region`.
6. Use **Dictionary** to inspect field metadata.
7. Build dashboards, reports, pivot tables and statistics.
8. Export PNG, CSV, HTML report or workspace JSON.

No AI API, backend, database or paid service is required.
